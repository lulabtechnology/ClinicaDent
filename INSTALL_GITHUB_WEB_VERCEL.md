# INSTALL_GITHUB_WEB_VERCEL.md

## Subir usando GitHub Web

1. Descarga y descomprime este ZIP.
2. Crea un repositorio nuevo en GitHub.
3. Entra al repositorio.
4. Haz clic en **Add file**.
5. Luego en **Upload files**.
6. Arrastra todo el contenido del proyecto.
7. Espera que termine de subir.
8. Haz clic en **Commit changes**.

## Conectar en Vercel

1. Entra a Vercel.
2. Haz clic en **Add New**.
3. Luego en **Project**.
4. Busca el repositorio.
5. Haz clic en **Import**.
6. Deja la configuración por defecto.
7. Haz clic en **Deploy**.
