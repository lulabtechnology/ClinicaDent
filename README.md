# ORTHOCLINIX Landing Page

Proyecto completo de una landing premium para ORTHOCLINIX, construido con Next.js, TypeScript, Tailwind CSS y Framer Motion.

## Qué contiene

- Landing page única orientada a conversión
- Header con anchor links
- Hero responsive con imagen desktop y mobile
- 2 secciones de video
- 8 imágenes de galería
- Sección de experiencia
- Servicios principales
- Diferenciales
- Proceso
- Precios
- FAQ
- CTA final
- Footer
- Sticky WhatsApp en mobile

## Archivos clave para editar

- `lib/site.ts`
- `app/page.tsx`
